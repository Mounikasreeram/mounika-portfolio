# Mounika Sreeram Portfolio

Personal portfolio for Mounika Sreeram, Senior Frontend Engineer.

## Tech Stack

- Next.js
- React
- TypeScript-oriented portfolio content
- Tailwind CSS
- Static portfolio assistant chatbot

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```

## Content Files

- `utils/data/personal-data.js` - name, photo, resume, contact links
- `utils/data/experience.js` - work experience
- `utils/data/projects-data.js` - project highlights
- `utils/data/skills.js` - skills shown in the marquee
- `public/profile.png` - profile photo
- `public/resume.pdf` - downloadable resume
- `app/components/portfolio-chatbot.jsx` - static portfolio chatbot
